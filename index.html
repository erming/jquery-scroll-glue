Usage

Include scripts

Start by including jQuery and the plugin.

<script src="jquery.js"></script>
<script src="jquery.scrollGlue.js"></script>

Active the plugin

Now to the easy part.. All you have to do is call .scrollGlue() on your elements.

<script>
	// Put the function call inside the
	// jQuery ready event:
	$(function() {
		$('.container').scrollGlue({
			speed: 0 // The scroll animation speed
		});
	});
</script>

<style type="text/css">
.container {
	// This is just an example use case
	overflow-y: scroll;
}
</style>

<div class="container">
	// A scrollable container
</div>
